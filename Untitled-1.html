Model Performance & Evaluation

  • Dataset: Twitter US Airline Sentiment dataset (airline_sentiment.csv) with 14K+ labeled tweets (Positive, Negative, Neutral).
  • Model: Many-to-One LSTM trained for Sentiment Classification.
  • Training: Preprocessed text using tokenization, padding, and embeddings; trained with categorical cross-entropy loss and Adam optimizer.

    • Key Results

  • Accuracy: ~88% on validation set
  • F1-Score (Macro): ~0.87
  • Precision / Recall: Balanced across positive and negative classes; slightly lower recall for neutral due to class imbalance.
  • Confusion Matrix: Shows strong classification for Positive/Negative, minor misclassifications in Neutral.

    • KPIs Used

  • Loss vs Epochs & Accuracy vs Epochs (to measure convergence and avoid overfitting).
  • F1-score (to evaluate performance across imbalanced classes).
  • Training Time per Epoch (efficiency check).
  • Text Generation Quality (evaluated qualitatively using generated text from alice.txt).
